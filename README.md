# IR-Sensor

## Objective
Create a low-power infrared sensor module that can be used with an ATtiny microcontroller using I2C. 

## Components
ATtiny Microcontroller
Infrared LED
Photodiode
MCP6002 Operational Amplifier
Resistors and Capactiors
